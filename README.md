# 🛍️ Venzora Shopping Mall

A modern and responsive shopping website built using **HTML, CSS, and JavaScript**.  
This project provides an attractive e-commerce UI with product filtering, searching, cart functionality, wishlist support, and smooth shopping experience.

---

## ✨ Features

- 🏠 Beautiful Hero Section
- 🔍 Product Search Functionality
- 📂 Category Filtering
- ❤️ Wishlist / Favorite Products
- 🛒 Add to Cart System
- 💳 Checkout Functionality
- 💰 Automatic Discount Calculation
- 📱 Responsive Shopping Layout
- 🎨 Modern UI with Animations
- 💾 Local Storage Cart Saving

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

---

## 📂 Product Categories

- Electronics
- Men Fashion
- Women Fashion
- Kids Collection
- Home Decor

## 🚀 How It Works

1. Browse products
2. Filter by category
3. Search products instantly
4. Add items to cart
5. View total bill and discount
6. Checkout successfully

---

## 💡 Main Functionalities

### 🔎 Search Products
Users can search products dynamically using the search bar.

### 📂 Filter Products
Products can be filtered category-wise:
- Electronics
- Men
- Women
- Kids
- Decor

### 🛒 Cart System
- Add multiple products
- Quantity updates automatically
- Cart data stored using Local Storage

### 💸 Discount Feature
- 10% discount applied automatically for orders above ₹5000

### ❤️ Wishlist Feature
Users can mark favorite products using heart icons.
